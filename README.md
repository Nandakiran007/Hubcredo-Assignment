# MERN Auth Minimal Project

Two folders at project root:

-   `backend` — Node.js + Express + MongoDB
-   `frontend` — React + Vite

See individual READMEs in each folder for start instructions. Basic steps:

1. Start MongoDB (local or set `MONGO_URI` in `backend/.env`).
2. Backend: `cd backend` → copy `.env.example` to `.env` and set values → `npm install` → `npm run dev` or `npm start`.
3. Frontend: `cd frontend` → `npm install` → `npm run dev`.

Defaults:

-   Backend: `http://localhost:5000`
-   Frontend (Vite): `http://localhost:5173`
